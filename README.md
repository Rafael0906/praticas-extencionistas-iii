# Práticas Extensionistas III — Entrega 1

## Tema do projeto
**Sistema de Gestão de Ações Comunitárias e Doações**

Este repositório reúne a documentação inicial de modelagem do sistema de informação e do banco de dados, desenvolvida para a **Entrega 1** do projeto de **Práticas Extensionistas III**.

## Objetivo do sistema
O sistema tem como objetivo apoiar ações extensionistas por meio do cadastro e gerenciamento de:
- voluntários
- ações comunitárias
- beneficiários
- doações
- atendimentos realizados

A proposta é permitir melhor organização das atividades sociais, controle das doações, histórico de atendimento e acompanhamento dos participantes envolvidos em cada ação.

## Artefatos desta entrega
Foram desenvolvidos os seguintes diagramas:
1. Modelagem de Entidade e Relacionamento (Conceitual)
2. Modelagem de Entidade e Relacionamento (Lógico)
3. Diagrama de Classes
4. Diagrama de Caso de Uso Geral
5. Diagrama de Sequência
6. Diagrama de Atividades — Cadastro de ação comunitária
7. Diagrama de Atividades — Atendimento ao beneficiário

## Descrição resumida dos diagramas

### 1. MER Conceitual
Representa as principais entidades do domínio: voluntário, ação comunitária, doação e beneficiário, bem como seus relacionamentos.

### 2. MER Lógico
Apresenta as tabelas do banco de dados, com chaves primárias e estrangeiras, prontas para implementação.

### 3. Diagrama de Classes
Mostra a estrutura orientada a objetos do sistema, com atributos e métodos principais das classes centrais.

### 4. Diagrama de Caso de Uso
Evidencia os atores do sistema e as funcionalidades principais disponíveis para cada perfil.

### 5. Diagrama de Sequência
Descreve o fluxo de registro de uma doação e sua vinculação a um beneficiário.

### 6. Diagrama de Atividades — Cadastro de ação
Apresenta o passo a passo do fluxo de cadastro, validação e publicação de uma ação comunitária.

### 7. Diagrama de Atividades — Atendimento
Demonstra o processo de solicitação, identificação do beneficiário e registro do atendimento.
